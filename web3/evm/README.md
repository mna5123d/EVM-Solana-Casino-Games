# EVM Smart Contracts

This directory contains Solidity contracts for all casino games, compatible with Ethereum, Polygon, Base, Arbitrum, and other EVM chains.

## Planned Structure

```
src/
├── Crash.sol         ← Crash game contract
├── CoinFlip.sol      ← Coinflip contract  
├── Plinko.sol        ← Plinko contract
├── Dice.sol          ← Dice contract
├── interfaces/       ← Contract interfaces
└── libraries/        ← Shared libraries
```

## Framework

Using Foundry for development, testing, and deployment.
