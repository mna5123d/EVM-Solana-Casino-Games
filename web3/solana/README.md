# Solana Smart Contracts

This directory contains Solana programs for all casino games built with Rust and Anchor framework.

## Structure

```
programs/
├── crash/          ← Crash game logic
├── coinflip/       ← Coinflip game logic
├── plinko/         ← Plinko game logic
├── dice/           ← Dice game logic
├── jackpot/        ← Jackpot pool logic
└── common/         ← Shared utilities
```
